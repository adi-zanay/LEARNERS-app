CAPACITOR CAPACITOR Build native mobile apps with web technology and Angular Portal for Capacitor The Power of Capacitor

Installed nearly half a million times per month and growing, Capacitor is Ionic’s solution to bridge web and native applications, providing seamless mobile experiences to users around the world. Since the release of Capacitor in May 2019, thousands of companies have used the cross-platform native runtime to build web apps that run on iOS, Android, desktop, and on the web as Progressive Web Apps.

 Building Capacitor apps at scale

Capacitor apps harness the power of the web to quickly build mobile applications. However, when developing large-scale Capacitor apps, as with any mobile application, team organization can limit the potential to move even faster. Here are some potential challenges large teams could face:


For new features to reach users, the entire application needs to be ready to ship at the same time. This lag can lead to bottlenecks in the development process and a slower shipping cadence.

Managing multiple developers can be difficult. When building Capacitor apps, a team of multiple developers—often spanning 10 or more developers—will likely work on the same code base.

Install Capacitor.

Add Capacitor to your project using the ng-app schematic

$ ng add capacitor/angular

Build the Web App.

The compiled web assets will be copied into each Capacitor native platform during the next step.

$ ng build --prod

Install the native platforms you want to target.
 

Capacitor's native projects exist in their own top-level folders and should be considered part of your app (check them into source control).

$ npm i @capacitor/ios @capacitor/android
$ npx cap add android
$ npx cap add ios
